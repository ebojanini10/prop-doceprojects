# prop-doceprojects

Propuestas comerciales cifradas de DoceProjects (12:12 Projects).

**URL:** https://prop.doceprojects.com
**Repo:** `ebojanini10/prop-doceprojects`
**Dir local:** `infrastructure/prop-doceprojects/`

## Deploy

```bash
cd infrastructure/prop-doceprojects
./deploy.sh <client-slug> <ruta/propuesta.html>
# Password por defecto: doce-<client-slug>
```
